## 🛠️ 开发工具推荐

### 1. Claude Code
**简介**：Anthropic 推出的官方 CLI 工具  
**特点**：
- 深度理解代码上下文
- 支持复杂的重构任务
- 可以直接执行 shell 命令  
**适用场景**：需要深度代码理解和复杂操作的场景

### 2. Obsidian
**简介**：个人知识管理工具  
**特点**：
- 本地存储，数据安全
- 强大的双向链接功能
- 丰富的插件生态  
**适用场景**：知识管理、笔记整理、写作素材收集
